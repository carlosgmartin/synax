.. py:class:: synax.Linear(input_dim: int, output_dim: int, initializer: jax.nn.initializers.Initializer = nn.initializers.he_normal(), regularizer: synax._regularizers.Regularizer = zero)

   Linear map.

   Does not include bias.

   Computes

   .. math::
       y = A x

   where :math:`A` is a learned matrix.

   :param input_dim: Input dimension.
   :param output_dim: Output dimension.
   :param initializer: Initializer.
   :param regularizer: Regularizer.
